表情API


表情识别


从一张图像中识别一个或者多个人的表情，同时也返回人脸的位置。 可识别的情绪包括开心，伤心，自然，惊恐四种。


注意:

  • 输入图像文件类型包括JPEG，PNG，GIF，BMP，图像尺寸小于4M

  • 可识别的人脸像素大小在36×36和4096×4096之间，若人脸尺寸不在 这个范围内，将不会被检测到。

  • 有些人脸由于技术性问题无法检测出来。 例如：奇怪的Poss,巨大的人脸之类


提供服务的方式：

应用程序向服务器端Post信息，服务器返回Json串， Post请求的Json格式如下:


Request URL


http://westus.api.hyr.com/emotion


Request headers


  • Content-Type

    内容类型。(在线url或本地图片)



  • Ocp-Apim-Subscription-Key

    应用密钥，购买后显示在个人中心



Request body

  • application/json

    { "url": "http://example.com/picture.jpg" }


  • application/octet-stream

    二进制文件



服务器返回值：


Response 200


  • 成功返回，Json串格式如下：

    [
  {
    "faceRectangle": {
      "left": 68,
      "top": 97,
      "width": 64,
      "height": 97
	},
    "scores": {
      "happiness": 0.9875571,
      "neutral": 0.0009861537,
      "surprise": 0.008229999,
      "sadness":0.0015
    }
  }
]

Response 400：


  • Json串解析错误


Response 401：


  • APP Key错误，请到个人中心检查


Response 409：


  • 超过对应套餐限定额


例子：

  • Python2.7(其余语言类似)

    ########### Python 2.7 #############
import httplib, urllib, base64

headers = {
    # Request headers
    'Content-Type': 'application/json',
    'Ocp-Apim-Subscription-Key': '{subscription key}',
}

params = urllib.urlencode({
})

try:
    #ulr is mentioned above
    conn = httplib.HTTPSConnection('{url}')
    conn.request("POST", "/emotion" % params, "{body}", headers)
    response = conn.getresponse()
    data = response.read()
    print(data)
    conn.close()
except Exception as e:
    print("[Errno {0}] {1}".format(e.errno, e.strerror))
